Development
-----------
The site can be served locally using _jekyll_. _jekyll_ can be obtained as a _ruby_ _gem_

### Pre-requisites
```zsh
pacman -S ruby #or apt-get install ruby
gem install jekyll yajl-ruby
```

### Local deploy
```zsh
cd path/to/project
jekyll serve --watch
```
